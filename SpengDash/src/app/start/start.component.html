SpengDash ist eine von Schülern umgesetzte Seite, die alle wesentlichen Informationen auf einer Website vereint. Auf der SpengDash-Seite können sowohl Schüler als auch Lehrer der HTL-Spengergasse die für den Alltag relevanten Informationen einsehen.
Dies beinhaltet folgende Daten:

  • Schularbeits / Test Termine
  • Aufgaben
  • Stundenplan
  • Freie Räume
  • Freie Tage
